## Quick Install

1. In WordPress admin, go to `Plugins > Add New > Upload Plugin`.
2. Upload `plugin-zip/smart-checkout-enhancer.zip`.
3. Activate plugin.
4. Configure at `WooCommerce > Settings > Smart Checkout Enhancer`.
